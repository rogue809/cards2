# 学习笔记 PWA 应用

这是一个渐进式Web应用(PWA)，用于管理学习笔记、单词卡片和复习计划。

## 功能特点

- 创建和管理学习笔记
- 添加图片到笔记
- 按标签分类笔记
- 设置掌握程度
- 复习功能
- 数据备份和恢复
- 离线使用

## 部署说明

1. 将项目上传到GitHub仓库
2. 在仓库设置中启用GitHub Pages
3. 选择主分支作为发布源

## 安装到主屏幕

- 在Android Chrome中：点击菜单 > "添加到主屏幕"
- 在iOS Safari中：点击分享图标 > "添加到主屏幕"

## 开发

1. 克隆仓库
2. 打开index.html进行开发
3. 提交更改并推送到GitHub
